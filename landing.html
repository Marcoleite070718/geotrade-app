Dúvidas comuns
Os cálculos são confiáveis para uso real? +
Sim. O simulador usa as alíquotas vigentes da TEC e o método de ICMS por dentro conforme a legislação brasileira. O câmbio é obtido em tempo real via API do Banco Central. Para a DI oficial, sempre valide com seu despachante — o GeoTrade é uma ferramenta de planejamento e negociação, não substitui a DI.
Preciso de CNPJ para usar? +
Não. O GeoTrade funciona com login pelo Google. Qualquer pessoa pode usar o plano Free imediatamente, sem cadastro de empresa.
O que é o ICMS por dentro e por que importa? +
Na importação, o ICMS é calculado sobre uma base majorada que inclui o próprio ICMS — o chamado cálculo "por dentro". A maioria das planilhas usa o cálculo "por fora", que subestima o imposto em até 22%. O GeoTrade usa o método correto conforme a legislação, para você não ter surpresas na DI.
Como funciona o plano Free? +
O Free permite 5 simulações por mês sem custo. É suficiente para avaliar o produto e testar operações pontuais. Para importadores que simulam semanalmente, o Starter é o mais indicado.
Posso cancelar quando quiser? +
Sim. Não há contrato de fidelidade. O cancelamento é imediato e você continua com acesso até o fim do período pago.
O GeoTrade cobre ex-tarifários e Drawback? +
Sim. O módulo de Regimes Especiais explica ex-tarifário, Drawback, RECOF, Admissão Temporária e Entreposto Aduaneiro — com critérios de elegibilidade e como aplicar em cada situação.
