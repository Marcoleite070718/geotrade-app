Dúvidas comuns
Os cálculos são confiáveis para uso real? +
Sim. O simulador usa as alíquotas vigentes da TEC (Tarifa Externa Comum) e o método de ICMS por dentro conforme a legislação brasileira. O câmbio é obtido em tempo real via API do Banco Central. Para declaração aduaneira, sempre valide com seu despachante — o GeoTrade é uma ferramenta de planejamento e negociação, não substitui a DI.
Preciso de CNPJ para usar? +
Não. O GeoTrade funciona com login pelo Google. Qualquer pessoa pode usar o plano Free imediatamente, sem cadastro de empresa.
Como funciona o plano Free? +
O Free permite 5 simulações por mês sem custo. É suficiente para avaliar o produto e testar operações pontuais. Para importadores que simulam semanalmente, o Starter é o mais indicado.
O GeoTrade calcula ICMS por dentro? +
Sim — e este é um dos principais diferenciais. A maioria das planilhas usa a alíquota simples, subestimando o ICMS. O GeoTrade usa o cálculo correto "por dentro" (base de cálculo majorada), que é o método exigido na importação. A diferença pode chegar a 22% no valor final do imposto.
Posso cancelar quando quiser? +
Sim. Não há contrato de fidelidade. O cancelamento é imediato e você continua com acesso até o fim do período pago.
